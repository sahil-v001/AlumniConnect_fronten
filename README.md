## 🚀 Overview

**AlumniConnect** is a centralized web-based platform designed to manage alumni data and enhance engagement between alumni and institutions.

This repository contains the **frontend implementation**, built using modern technologies to deliver a fast, responsive, and user-friendly experience.

---

## ✨ Features

- 👤 User profile interface  
- 🔍 Alumni search functionality  
- 📅 Event browsing interface  
- 🤝 Mentorship interaction UI  
- ⚡ Fast and responsive design  
- 🎨 Clean and modern UI  

---

## 🛠️ Tech Stack

- **React.js**  
- **Vite**  
- **Tailwind CSS**  
- **JavaScript (ES6+)**  
- **PostCSS**
